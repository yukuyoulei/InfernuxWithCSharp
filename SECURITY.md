# Security Policy

## Supported versions

Infernux is currently in active technical-preview development. Security fixes are applied on the latest development line first.

## Reporting a vulnerability

Do not open a public issue for an unpatched security problem.

Please report vulnerabilities by email to `chenlizheme@outlook.com` with:

- a clear description of the issue
- affected version or commit information
- reproduction steps or a proof of concept
- impact assessment if known

If the report is valid, the goal is to acknowledge it quickly and follow up with remediation planning and disclosure timing.

## Scope notes

This repository includes native runtime code, Python tooling, packaging workflows, and a website/docs surface. Please mention which area is affected when reporting an issue.
