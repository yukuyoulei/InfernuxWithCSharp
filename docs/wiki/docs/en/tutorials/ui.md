# Building UI in Infernux

The current UI layer provides the fundamentals needed for in-game interfaces: [UICanvas](../api/UICanvas.md), [UIText](../api/UIText.md), [UIImage](../api/UIImage.md), [UIButton](../api/UIButton.md), and [UISelectable](../api/UISelectable.md).

## Basic composition

1. Create a canvas root.
2. Add text and image elements for the visual layout.
3. Add buttons for interaction.
4. Handle events through button callbacks or pointer event data.
5. Drive UI state from gameplay or controller components.
