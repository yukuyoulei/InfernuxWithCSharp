# Infernux UI 入门

当前 UI 系统已经提供基础界面能力：[UICanvas](../api/UICanvas.md)、[UIText](../api/UIText.md)、[UIImage](../api/UIImage.md)、[UIButton](../api/UIButton.md) 和 [UISelectable](../api/UISelectable.md)。

## 基本搭建步骤

1. 创建 Canvas 根节点。
2. 添加文本和图片元素。
3. 用按钮承载交互。
4. 通过按钮回调或指针事件数据处理输入。
5. 让玩法脚本驱动 UI 状态。
